# 微观经济学课程资料

This repository hosts course materials for microeconomics, including lecture notes, handouts, and supplementary PDFs.

## 内容 Contents

- `2025/`：2025 学年使用的讲义与附录
- `2026/`：2026 学年的讲义、补充讲义与课程相关总结材料

## 使用方式 Usage

学生可以直接在 GitHub 在线预览 PDF，也可以下载后离线阅读。

Students can preview files directly on GitHub or download the PDFs for offline reading.

## 说明 Note

这些资料用于教学与学习交流；如需转载或二次分发，请先联系授课教师。

These materials are shared for teaching and learning purposes. Please contact the instructor before redistributing them elsewhere.
